Projet 1 : Portfolio Wordpress

Partie 1 : Conception d’un site web WordPress

Cahier des charges :

    Concevoir un site web qui sera composé de :

  • Un onglet « Accueil » ⇒ présentation du BTS SIO SLAM, liens, photos,
  • Un onglet « CV » ⇒ CV en HTML/CSS, CV téléchargeable en version PDF,
  • Un onglet « ÉCOLE » ⇒ présentation du CFA INSTA, lien du site, photos,
  • Un onglet « Entreprise » ⇒ présentation de l’entreprise, lien du site, photos,
  • Un onglet « Cours » ⇒ premier chapitre + série d’exercices + corrections,
  • Un onglet « Contact » ⇒ formulaire de contact (nom, prénom, e-mail, sujet)

Spécifications

  • Utiliser « TRELLO » ⇒ gestion du projet (tâches),
  • Utiliser un « diagramme de Gantt » ⇒ gestion du temps,
  • Utiliser le Adobe XD pour la maquette et CMS WordPress pour la réalisation,
  • Utiliser un Server local (type Wamp), ou hébergement web (type OVH),
  • Créer un PowerPoint de présentation.

    Projet 2 : Création d'un site de gestion de base de données (PHP/MySQL)

    Cahier des charges :

    Le site Garage a été réalisé dans le cadre d’un travail dirigé à la fin d’un chapitre de cours sur PHP et la gestion des Bases de données avec MySQL. Le projet consistait à créer un site qui permet la gestion d’un garage.

    • Pour se faire, nous devions utiliser les langages PHP, JS, HTML5 et CSS3, le Framework Bootstrap, et l’SGBD MySQL.
    • Nous devions inclure quatre tables Client, Véhicule, Technicien, et Intervention.
    • Le site devait permettre une connexion / déconnexion avec des rôles (User et Admin).
    • Le site devait avoir un menu de navigation avec les onglets Home, Client, Véhicule, Technicien, Intervention.
    • Enfin, il devait également avoir un pied de page avec des informations utiles et des contacts.
    • Il devait permettre les actions usuelles sur les tables : Ajouter, Lister, Supprimer, Modifier.
